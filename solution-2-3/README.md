# Trace — Evidence graph

An editable PlateJS audit memo with an evidence-graph method for following connected records and spotting missing links.

## Demo

- Four dossier-grounded finding paths
- Switchable user, vendor, document, payment, policy, and missing-edge nodes
- Fully functional **Add case** flow with linked source passages
- Inline report citations and a compact Sources page
- Draggable exact-source windows
- Three cleared reconciliation checks

## Run

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 42933
```

## Verify

```bash
npm run lint
npm run build
```
