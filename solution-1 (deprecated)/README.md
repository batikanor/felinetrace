# Trace — audit desk

Editable PlateJS working paper for the Muster Verpackungen 2025 dossier.

## Demo

- 27 business files in one source ribbon
- Four review items with exact row, cell, section, and page links
- Draggable source windows
- Three passed checks to show false-positive control
- Formatting and comments

## Run

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 41827
```

## Build

```bash
npm run build
```
