# Trace — citations

Editable PlateJS working paper with numbered evidence links and a compact Sources page.

## Demo

- Four dossier-grounded review items
- Inline `[1] [2]` evidence links
- Exact row, cell, section, and page anchors
- Draggable source windows
- Three passed checks

## Run

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 42832
```

## Build

```bash
npm run build
```
