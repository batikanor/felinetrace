# FelineTrace — adaptive router

The cited PlateJS audit memo with a cost-aware method router and manual case linking.

## Method

1. Run deterministic ledger and control tests.
2. Compare robust peer groups and transaction sequences.
3. Traverse dossier-scoped evidence memory when relationships matter.
4. Check public sources only for entity corroboration.
5. Send unresolved conflicts to a local, read-only reviewer.

Every reported claim still requires an exact dossier passage. The router holds single-signal leads instead of turning every anomaly into a finding.

Use the **Setup** tab for local readiness, Codex authentication, optional credentials, and adapter health checks.

## Run

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 42999
```

## Verify

```bash
npm run lint
npm run build
```
