# 2-9 and 2-10 UX variants

All variants keep the editable PlateJS report, evidence links, source windows, manual case flow, and Setup page.

## Adaptive router

| Folder | Top workspace |
| --- | --- |
| `solution-2-9-2` | Finding drawer: cases left, selected route right |
| `solution-2-9-3` | Route form: roomy case choices and method fields |
| `solution-2-9-4` | Stage accordion: method stages first, selected case beside them |
| `solution-2-9-5` | Focus navigator: compact case tabs and one large route canvas |

## Claim Compiler

| Folder | Top workspace |
| --- | --- |
| `solution-2-10-2` | Claim form: claim selection followed by five proof fields |
| `solution-2-10-3` | Claim accordion: vertical claims and stacked proof gates |
| `solution-2-10-4` | Proof checklist: gates left, evidence and certificate right |
| `solution-2-10-5` | Review wizard: stage strip with a large current proof area |

Setup uses one status per integration. The status shown beneath each Test button is the test result.
