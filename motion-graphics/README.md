## Direction

- The screen supplies the proof; the speaker supplies the explanation.
- Avoid captions, paragraphs, and redundant callouts.
- Hold amounts and exact locations long enough to be read.
- If sound is added, keep it structural: soft data ticks, link clicks, and one confirmation tone.
