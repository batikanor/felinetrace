# Trace — evidence trace

Editable PlateJS working paper with one focused evidence chain at a time.

## Demo

- Source → item → report-section trace
- Four dossier-grounded review items
- Exact row, cell, section, and page anchors
- Draggable source windows
- Three passed checks

## Run

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 43843
```

## Build

npm run build
