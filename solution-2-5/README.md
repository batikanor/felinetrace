# FelineTrace — benchmark studio

The cited PlateJS audit memo with a four-method ensemble and manual case linking.

## Method

- Deterministic audit rules
- Evidence graph paths
- Sequence and timing tests
- Counterevidence checks
- Source-backed consensus matrix

Users can add a manual case, describe it, select evidence passages, and reopen every linked source from the saved case.

## Run

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 42955
```

## Verify

```bash
npm run lint
npm run build
```
