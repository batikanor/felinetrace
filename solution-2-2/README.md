# Trace — Audit Test Lab

Deterministic audit tests connected to an editable PlateJS working paper.

## Demo

- Ingest → reconcile → test → countercheck → cite
- Coverage matrix for four dossier-grounded tests
- Editable report with 14 inline evidence links
- Draggable exact-source windows and compact Sources page
- Add cases with linked dossier passages

## Run

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 42922
```

## Verify

```bash
npm run lint
npm run build
```
