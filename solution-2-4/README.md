# Trace — Detector + Skeptic

Editable PlateJS working paper with cited findings and a visible countercheck method.

## Demo

- Claim → evidence → counterevidence search → verdict
- Direct-match rule before any offset reduces a case
- Four dossier-grounded cases and three cleared checks
- Inline citations, Sources page, and draggable exact-source windows
- Add-case form with source multi-select and skeptic status

## Run

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 42944
```

## Verify

```bash
npm run lint
npm run build
```
