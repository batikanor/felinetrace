# Trace — review margin

Editable PlateJS working paper with paragraph-linked evidence in the margin.

## Demo

- Select a report section to focus its evidence
- Four dossier-grounded review items
- Exact row, cell, section, and page anchors
- Draggable source windows
- Three passed checks and comments

## Run

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 44854
```

## Build

```bash
npm run build
```
